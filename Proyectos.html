Gestion de Herramientas Proyecto para Github
Proyecto realizado en lenguaje de programacion PYTHON.Realizado con la finalidad de ofrecerle un sistema de gestion de herramientas a los residentes de un conjunto de Edificios
Hay dos roles: Administrador y Usuario.
El rol de administrador permite: Agregar, Eliminar o Actualizar nuevos usuarios, Agregar,Eliminar o Actualizar Herramientas, Consultar PQRS realizadas por los usuarios, consultar los errores comunes que cometen los usuarios al solicitar una herramienta o ingresar a la plataforma. El rol de Administrador esta protegido con contraseña.
El rol de Usuario permite: Solicitar o Devolver una herramienta. Consultar prestamos realizados, Realizar una PQRS
El proyecto tiene permanencia de datos en archivos json. 		Proyecto realizado con la finalidad de mostrar el dominio en Git y Github.
